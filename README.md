Aplicativo para venda de microserviços.

-   Microsseguros
-   Plano de saúde
-   Aconselhamento médico
-   Capitalização

### Iniciar o projeto

> yarn

para utilizar no emulador ou no aparelho

-   IOS
    > yarn Ios
-   Android
    > yarn android
